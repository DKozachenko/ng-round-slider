# ng-round-slider

Angular library with implementation of [Round Slider JQuery Plugin](https://roundsliderui.com) using [JQuery](https://www.npmjs.com/package/jquery) and [round-slider](https://www.npmjs.com/package/round-slider) packages.

## Motivation

Sometime ago I needed a round slider as a map control. I didn't find a suitable for me vanilla (html, css, js) solution (but I googled about 2 hours). Only option was as mentioned earlier [JQuery Plugin](https://roundsliderui.com). I integrated it to my Angular application. I dreamed it would be a separate library I could reuse...

## Usage

1. Install library:

```bash
npm i ng-round-slider
```

2. Import the component:

```typescript
// ...
import { NgRoundSliderComponent } from 'ng-round-slider';

@Component({
  // ...
  imports: [NgRoundSliderComponent]
})
export class SomeComponent {
  // ...
}
```

3. Add in template:

```html
<ng-round-slider></ng-round-slider>
```
